## Description

This skill provides a workflow for effectively using typography in digital design. It emphasizes type hierarchy, readability, appropriate font pairing, and performance optimization.

## Scope

- Personal skill for UI/UX designers and frontend developers
- Applicable to web and app design
- Can be used for design system creation and font implementation

## Workflow Steps

### 1. Type Hierarchy

- Define heading levels (h1-h6)
- Create size scale
- Use consistent proportions
- Create visual distinction
- Maintain readability at all sizes
- Use weight for emphasis
- Document hierarchy

### 2. Font Selection

- Choose appropriate typefaces
- Consider brand identity
- Select serif or sans-serif appropriately
- Ensure web font availability
- Test readability
- Consider accessibility
- Plan font files

### 3. Font Pairing

- Pair compatible fonts
- Create contrast between heading and body fonts
- Use monospace for code
- Maintain consistency
- Test combinations
- Consider cultural appropriateness
- Document pairings

### 4. Readability

- Use appropriate line length (50-75 characters)
- Set line height 1.5-1.6 for body text
- Use adequate font size (16px minimum)
- Ensure sufficient contrast
- Use appropriate line spacing
- Consider letter spacing
- Test readability

### 5. Web Font Implementation

- Use system fonts when appropriate
- Optimize web font loading
- Use font-display property
- Implement fallback fonts
- Load only necessary weights
- Use variable fonts if available
- Monitor performance impact

### 6. Responsive Typography

- Scale fonts appropriately
- Use scalable units (rem)
- Adjust at different breakpoints
- Consider mobile readability
- Test on various devices
- Maintain hierarchy
- Ensure legibility

### 7. Accessibility

- Use adequate font size
- Ensure sufficient contrast
- Avoid excessive styling
- Support text scaling
- Use semantic HTML
- Test with screen readers
- Support user preferences

### 8. Performance

- Minimize font files
- Use compression
- Implement font subsetting
- Consider loading strategy
- Monitor performance
- Test on slow connections
- Optimize critical path

## Quality Criteria

- Typography creates clear hierarchy
- Text is readable and accessible
- Fonts support brand identity
- Performance is optimized
- Responsive design works well
- Dark mode is supported
- Font strategy is documented
- Team can implement consistently

## Example Usage

When establishing typography system:

1. Select serif font for headings
2. Select sans-serif for body
3. Define size scale
4. Set line heights
5. Configure responsive scaling
6. Implement web fonts
7. Document font usage
8. Test accessibility
